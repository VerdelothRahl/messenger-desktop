Here's a simple unofficial desktop wrapper for facebook messenger.

🔥 Installing
1. Install Node.js if not already installed (https://nodejs.org/en/download)
2. Open a terminal in the project folder (shift right click in project folder - open in terminal)
3. Run npm install
4. Double-click Launch Messenger.lnk (or Launch Messenger.vbs)
